# RecoScout

RecoScout is a polished MVP web app for checking whether answer engines recommend a business or its competitors for buyer-intent questions.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Local mock audit data

## Run locally

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:3000`.

## Checks

```bash
npm run lint
npm run build
```
